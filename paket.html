FAQ

Pertanyaan yang sering ditanyakan

Deluxe menawarkan hotel AC + air panas, porter 1:1, matras 8cm, makanan lengkap dengan buah & 3L air mineral/hari, bonus makan malam & guide air terjun. Tidak ada biaya tambahan apapun. Standard/Sharing menawarkan hotel lebih sederhana, porter 1:2, makanan standar. Ada biaya tambahan jika kembali sebelum program selesai.

Ya, harga untuk Via Senaru dan Via Sembalun adalah sama. Rute dipilih saat proses booking. Untuk program 3D2N ke atas, tersedia opsi one-way Sembalun → Senaru tanpa biaya tambahan.

Untuk paket Standard/Sharing, jika tidak sampai puncak dan ingin kembali sebelum program selesai, ada biaya kompensasi untuk porter. Paket Deluxe tidak dikenakan biaya tambahan apapun — termasuk jika program tidak selesai dari Sembalun.
